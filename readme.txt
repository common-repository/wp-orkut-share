== Description ==

WP Orkut Share allows website owners to enable their visitors to share the website content with visitor's orkut friends. Users can optionally promote the content among their friends.

Features
-----------------------
1) You can choose the pages where you want to show the Orkut Share button.
2) You can set the languges supported by google orkut share (English, French, German, Portuguese (Brazil)).
3) You can set the post image thumbnail meta name and that image will be selected as thumbnail image for sharing.
4) If no image thumbnail meta name is set, first image in that post will be selected as thumbnail image for sharing.

== Installation ==

1. Upload "wp-orkut-share" directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto settings "Orkut Share Settings" and save your settings
